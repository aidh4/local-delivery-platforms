### 常见问题 FAQ

- 不同平台在外卖配送的时效差距如何判断？
  用同一地址与时段对比 ETA、忙时加价与历史评分，连续抽样 3–5 次，结合实际到达偏差评估稳定性。

- 想覆盖生鲜杂货与酒水外送，如何搭配平台？
  以综合平台+品类型平台组合：餐饮用 Postmates/Uber Eats/Doordash，杂货选 Instacart/Shipt，便利急需用 Gopuff，减少缺货与延迟。

- 如何控制同城即时配送的总成本？
  关注配送费、小额订单加价与忙时价格；尽量合并下单或选择预约时段，使用订阅制减免配送费用，形成固定周期开单习惯。
